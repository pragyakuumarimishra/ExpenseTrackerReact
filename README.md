# PExpense Pro 💸
A multi-user cloud-based expense tracker with Firebase Auth and Firestore.

### Features:
- 🔐 Login / Signup (Firebase)
- 📊 Chart-based analytics
- ☁️ Real-time database
- 🔒 Secure per-user storage

### Tech Stack:
- React
- Firebase (Auth + Firestore)
- Chart.js
